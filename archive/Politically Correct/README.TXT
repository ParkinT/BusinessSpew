Included in this package is the standard PC library.
But this program was designed to be "extensible".
That means YOU can create your own specialized libraries.
Think about a custom library for any particular trade or industry.
Every business has its own "esoteric" language that can be applied for comic results.

So, also included is the custom.lib file for you to build your own library(s).
Click on Process from the menu of PC and choose Select Library File to choose this one.
If you wish to create more, just copy the file custom.lib to any other name!

A few simple cautions when creating libraries.  Be careful of words that can be used as nouns and verbs.
Words like bear, presence, work will not always translate as you expect.
This release of PC does not have the sophisticated logic required to take clues from the "context" of the sentence.
In fact, this initial release will only process individual words (not phrases)
If we receive the enough feedback from users, development will continue on more AI logic for these functions.
